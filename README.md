# 🐧 Kubuntu Profile

Мои конфигурационные файлы (dotfiles) для Kubuntu: Zsh, Oh My Zsh, WezTerm и другие настройки.

## 🚀 Установка

1. **Клонируйте репозиторий**  
   ```bash
   git clone git@github.com:gumirus/kubuntuProfile.git ~/.dotfiles
   ```

2. **Запустите установочный скрипт**  
   ```bash
   cd ~/.dotfiles
   ./install.sh
   ```

3. **Перезапустите терминал** или выполните `source ~/.zshrc`, чтобы изменения вступили в силу.

## 📦 Что установит скрипт

При запуске `./install.sh` будет автоматически выполнено:

| Компонент | Описание | Примечание |
|-----------|----------|------------|
| `curl`, `git`, `wget` | Базовые утилиты | Устанавливаются, если отсутствуют |
| `zsh` | Оболочка Zsh | Устанавливается, но не меняется оболочка по умолчанию |
| Oh My Zsh | Фреймворк для управления Zsh | Устанавливается в `~/.oh-my-zsh` |
| `zsh-syntax-highlighting` | Подсветка синтаксиса | Плагин для Zsh |
| `zsh-autosuggestions` | Автоподсказки команд | Плагин для Zsh |
| `wezterm` | Терминал | Опционально, запрашивается подтверждение |
| Docker (CE) | Платформа контейнеров | Устанавливается по официальной инструкции; после установки **нужна перезагрузка** |
| MesloLGS Nerd Font | Шрифт с иконками | Скачивается и устанавливается; требуется выбрать в настройках терминала |
| Ваши dotfiles | Конфигурационные файлы | Создаются символические ссылки на файлы из репозитория |

## 🧪 Проверка работоспособности

После установки откройте новый терминал и выполните:

```bash
echo $ZSH_CUSTOM   # должно показывать ~/.oh-my-zsh/custom
```

Если всё настроено правильно, вы увидите свою привычную тему и плагины.
