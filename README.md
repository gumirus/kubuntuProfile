# Установка

1. Клонируйте репозиторий:
   ```bash
   git clone git@github.com:gumirus/kubuntuProfile.git ~/.dotfiles

2. Перейдите в папку и запустите установочный скрипт:

  ```cd ~/.dotfiles
  ```./install.sh

3. Перезапустите терминал (или выполните source ~/.zshrc), чтобы изменения вступили в силу.

## Зависимости

Перед установкой убедитесь, что в системе есть:

Zsh (можно установить командой sudo apt install zsh)

Oh My Zsh (устанавливается отдельно: sh -c "$(curl -fsSL https://raw.githubusercontent.com/ohmyzsh/ohmyzsh/master/tools/install.sh)")

WezTerm (по желанию, можно установить через sudo apt install wezterm)


## ⚠️ Важное предупреждение

Скрипт **не устанавливает** Oh My Zsh и сам Zsh – он только настраивает их. На новой системе эти компоненты нужно поставить заранее (или добавить их установку в скрипт, но тогда потребуется `sudo` и интерактивное подтверждение). Решайте сами, хотите ли вы автоматизировать и это.

Если вы добавите установку зависимостей в скрипт, не забудьте про проверку на уже установленные пакеты, чтобы не переустанавливать каждый раз.

## 🧪 Итог

Если скрипт написан аккуратно и протестирован, инструкция **точно будет работать**. После клонирования и запуска `./install.sh` вы получите полностью настроенную среду. Успехов! Если нужна помощь с доработкой скрипта – обращайтесь.
